Detailed Description

Структура данных, которая хранит в себе данные о квартирах.

Структура содержит данные о площади каждой квартиры, колличестве жильцов и номере каждой квартиры.

  • Parameters
    square_flatпеременная, которая хранит данные о площади квартиры
    count_personперменная, которая хранит данные о колличестве людей, живущих в этой квартире
    IDпеременная, которая обозначает номер квартиры, присваиватеся она автоматически, ибо в файле ее нет
