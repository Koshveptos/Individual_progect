Библиотека, которая читает 2 исходных файла (каждый имеет свой тип и структуру), и передает их в основное приложение.

Для передачи данных используются типы данных, определенные в бибилотеке, также в основном приложении необходимы массивы динамической памяти для хранения данных из файлов.

Некоторые ее функции нужно использовать в строго заданном порядке, которые соответствуют логике работы со структурой файлов.
