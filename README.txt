Задание, выолненное в рамках учебной дисциплины АиП(алгоритмизация и программирование)
имеет документацию в html-формате, выполненную с помощью doxywizard
тестирование функций с помощью assert и два разных проекта - клиентское приложение и статическую библиотеку
библиотека для чтения файлов определенное формата, главное приложение - для выполнения действий с данными

так же в клиентском приложении есть мои ответы на тесты- answers.txt, и правильные ответы -outputs.txt
Файлы с данные необходимо будет распаковать и поместить и клиентское приложение(01..02..03 нужно будет подбирать исходя из номера теста), так же следует проверить названия файлов в самом приложении
